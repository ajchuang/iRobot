README

DEVELOPMENT & RUNNING ENV
------------------------------------------------------------
1. Java SDK 1.7 (PATH is also configured)
2. If you have bash environment, it will be easier.


HOW TO RUN
------------------------------------------------------------
1.  With bash, just invoke ./run.sh. It will start cleaning, 
    compiling, running automatically with default params.
    
2.  If you want to run other params, you will need to type the command.
    'java PathPlanner hw4_world_and_obstacles_convex.txt hw4_start_goal.txt'
    

OUTPUT
------------------------------------------------------------
1.  a simple java frame will be displayed and show the path
    and obstacle information.
2.  a route.txt will be generated to show the shortest path info.
